    Drawing Baxter

    About

    Baxter can autonomously draw uploaded images on flat surfaces. It uses an OpenCV edge detection algorithm to get the outline of image figures. We created a customized holder for Baxter's gripper to use an EXPO marker to draw on a whiteboard. It will use reinforcement learning to improve its drawing skills and precision.
